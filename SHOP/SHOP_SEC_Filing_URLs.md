## How to Download

### Download Individual Files
Use curl with appropriate headers to download from SEC EDGAR:

```bash
# For 40-F annual reports (HTML/iXBRL format)
curl -L -o "SHOP_Annual_FY2023.htm" "https://www.sec.gov/ix?doc=/Archives/edgar/data/1594805/000159480524000007/shop-20231231.htm" \
  -H "User-Agent: financial-research-tool admin@example.com"

# For 6-K quarterly reports (HTML format)
curl -L -o "SHOP_Quarterly_Q3-2024_Financial_Statements.htm" "https://www.sec.gov/Archives/edgar/data/1594805/000159480524000090/exhibit991financialstateme.htm" \
  -H "User-Agent: financial-research-tool admin@example.com"
```

### Note About File Formats
- Most Shopify filings are available as **HTML/iXBRL** (inline XBRL) format, not PDF
- The iXBRL format contains the full financial statements with embedded structured data
- To save as readable format:
  - Open in browser and save as PDF, or
  - Use a tool to convert HTML to PDF, or
  - Extract the text content using pdftotext or similar tools
